Video & audio calls
for any website.

VidStem is a lightweight WebRTC toolkit you can drop into any page in a few lines. Peer-to-peer media, a clean UI, and a tiny server you can run anywhere.

Start a call See the code

Built to be built upon

Everything you need for real-time calls, nothing you don't.

Lightweight

A zero-dependency client SDK and a signaling server with a single dependency. Runs on a $5 VPS or a Raspberry Pi.

Embed in minutes

Add a <script type="module">, point it at your room, and you have a call. No accounts, no SDK keys.

Runs anywhere

Pure web standards on the client; Node.js on the server. Works across Windows, macOS, Linux, and every modern browser.

Private by design

Media flows directly between participants. Your server only relays connection setup - it never sees audio or video.

Clean, themeable UI

A polished call interface with mute, camera, and screen share - restyle it entirely with a few CSS variables.

Community-first

Readable code, thorough comments, and an MIT license. Fork it, extend it, and ship it your way.

Three lines to a live call

  1. Run the server. npm install && npm start starts signaling on port 8080.
  2. Drop in the UI. Import VidStemUI, give it a container and a room.
  3. Share the link. Anyone who opens the page and enters the same room is in the call.
Open the live demo 
<!-- 1. Add the stylesheet -->
<link rel="stylesheet" href="/client/src/vidstem.css">

<!-- 2. A box for the call -->
<div id="call" style="height:600px"></div>

<!-- 3. Start it -->
<script type="module">
  import { VidStemUI } from '/client/src/ui.js';

  new VidStemUI({
    container:  '#call',
    serverUrl:  'ws://localhost:8080',
    room:       'team-standup',
    displayName:'Alice',
  }).start();
</script>
